External links

Programming

t3x.org
Webpage of Nils M Holm. There are a lot of interesting references and original programs!
Mar's GitHub homepage
GitHub homepage of a friend of mine!
Suckless
In this site you can find a lot of minimal alternatives to many programs. Of this group of programmers I especially like st, the simple terminal emulator and dwm, a very simple tiled window manager for the X Window System.
Chuck Moore's webpage
The webpage of FORTH's inventor and I found there a lot of interesting things. A very concise version of the FORTH programming system named ColorForth is also introduced.
Forth Interest Group
This site is a good starting point to find informations about the Forth programming language. There are many useful references and literature from which you can learn about it.
CAT -V Harmful things
Here interesting topics of Computer Science are debated and discussed with peppered sarcasm.
Floodgap A site about very old stuff. Discovered when looking for information about the Gopher protocol. If you are interested you can connect to their gopherhole directly or through their http proxy.
